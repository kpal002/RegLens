# `data/cases/`

Curated demo variants with known mechanisms live here (added Saturday), for the
"annotation shrugs, RegLens explains" money-shot. Each case will pin a variant +
cell-type context + the expected TF / gene / trait so the end-to-end demo is
reproducible.

The bundled *synthetic* demo used by `reglens demo` (a fake contig + stub model,
not a real biological result) lives in `reglens/data/demo/`.
